# Vocal Raid Assistant
</div>

Vocal Raid Assistant adds sounds to important party and raid cooldowns tracked by **[MiniCC](https://addons.wago.io/addons/mini-cc)**.

Since Midnight made spell tracking significantly more complex, VRA no longer handles it on its own. MiniCC is a dedicated addon built specifically for this — so rather than duplicating that work, VRA simply subscribes to its API and focuses on what it does best: playing sounds.

## Note
VRA can only vocalize what MiniCC can see. Blizzard decides which spells are marked as "important" and therefore trackable by MiniCC.

# Requirements
- **[MiniCC](https://addons.wago.io/addons/mini-cc)** — must be installed and configured
- Choose and install a **Voicepack** from the list below
